# Articles

Long-term article files should live here as Markdown or MDX, one folder per
article slug.

Example:

```text
articles/
  fall-protection/
    fall-protection.mdx
  silica/
    silica-exposure-control.mdx
  first-aid/
    cardiac-arrest-on-site.mdx
```

The current prototype article data is in `src/wikiContent.js`.
