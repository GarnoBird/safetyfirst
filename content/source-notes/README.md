# Source Notes

Use this folder for source-by-source notes before drafting article text.

Each source note should include:

- Official source title and URL.
- Jurisdiction and publisher.
- Effective date or last checked date when available.
- Regulation part, guideline number, policy item, or code reference.
- Short paraphrased summary of what the source supports.
- Articles that rely on the source.

Do not store long copied passages from official or proprietary documents.
